# 🚀 Meu Portfólio Pessoal

Uma landing page simples, moderna e responsiva, desenvolvida apenas com **HTML5** e **CSS3** para apresentar meus projetos e minha trajetória profissional.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando apenas tecnologias web nativas, sem frameworks:
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização, layout responsivo (Flexbox/Grid) e animações.

---

## ✨ Funcionalidades

- 📱 **Design Responsivo:** Adapta-se a computadores, tablets e smartphones.
- 🎨 **Estilização Moderna:** Uso de variáveis CSS para cores e fontes limpas.
- 🔗 **Links Diretos:** Acesso rápido aos meus melhores projetos e redes sociais.

---

## 📂 Como Executar o Projeto

Para visualizar a página localmente na sua máquina, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone 
   ```
2. **Abra a pasta do projeto:**
   ```bash
   cd seu-repositorio
   ```
3. **Execute o arquivo:**
   - Basta dar um duplo clique no arquivo `index.html` para abrir no seu navegador.
